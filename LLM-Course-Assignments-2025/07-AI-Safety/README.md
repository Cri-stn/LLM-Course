# 方向07：人工智能安全综合作业

## 题目描述
对当前最强开源LLM进行红队攻击与防御研究，设计新型越狱方法并提出防御机制。

## 题目要求
- **数据准备**：收集/合成越狱prompt数据集（如AdvBench、GAR）。
- **模型选择**：针对LLaMA-3.1、Qwen-2.5等模型。
- **核心功能**：自动生成越狱prompt、评估攻击成功率、实现防御（如自卫prompt、过滤器）。
- **迭代优化**：进化攻击策略、强化防御，提升鲁棒性。
- **部署**：构建红队工具Demo。
- **评估**：攻击成功率（ASR）、防御后下降幅度。
